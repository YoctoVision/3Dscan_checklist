# 系统概述

- AOI Based 3D Scan System 是一款先进的 3D 视觉检测系统。该系统融合了人工智能算法，能够实现对产品外观缺陷种类及有无进行全方位、立体化的精准识别与判读。

## 核心价值与特点

### 全面精准的三维缺陷检测

- 系统的核心能力在于对产品进行高精度的三维扫描，并利用AI算法对各类外观缺陷（如划痕、凹陷、毛刺）、进行实时识别与分类。同时，系统精确记录和管理批次结果，形成完整的质量追溯档案。

### 高效灵活的工位与视野配置

- 系统原生支持多工位、多相机同步或异步扫描，能够从多个角度完整覆盖复杂产品的各个表面，确保检测无死角。其模块化工位配置方案，可根据不同产线布局与产品类型进行快速调整与部署。

### 稳定可靠的工业级性能

- 专为满足工业现场 7×24 小时连续稳定运行而设计，具备优异的抗干扰与长时间工作能力。单次完整扫描与分析可在秒级时间内完成，能满足高节拍生产线的实时检测需求。

### 无缝的数据集成与流程管控

- 提供实时的数据监控仪表板，集中展示检测状态、系统运行日志及关键指标。支持与工厂现有的MES、ERP等生产管理系统进行深度集成，自动上传检测结果、缺陷图像及批次数据，实现检测流程与生产管理的无缝衔接。
